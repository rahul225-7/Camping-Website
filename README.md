# YelpCamp

YelpCamp is a full-stack web application for discovering, reviewing, and sharing campgrounds. Users can browse listings, add new campgrounds, and leave reviews with ratings.

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- Bootstrap
- Passport.js

## Features
- Campground CRUD
- Reviews with ratings
- User authentication
- Flash messages
- Responsive UI

## Notes
Built for learning and practicing full-stack web development.
